Html

É a linguagem usada para descrever e definir o conteúdo de uma página Web em um formato estruturado.

CSS

Cascading Style Sheets (CSS) é um simples mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um doumento web. Em vez de colocar a formatação dentro do documento, o CSS cria um link para uma página que contém os estilos..

JavaScript

É usado para adicionar interatividade a uma página web e criar aplicações web ricas.

Json

JSON é um formato leve para intercâmbio de dados computacionais. JSON é um subconjunto da notação de objeto de JavaScript, mas seu uso não requer JavaScript exclusivamente.
